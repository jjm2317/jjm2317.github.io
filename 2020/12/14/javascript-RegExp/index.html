정규표현식

정의

일정한 패턴을 가진 문자열의 집합을 표현하기 위해 사용하는 형식언어

기능

  • 문자열을 대상으로 패턴 매칭 기능 제공
    • 특정 패턴과 일치하는 문자열 검색 / 추출 / 치환

  • 정규 표현식의 검색방식 설정

    • g
      • 패턴과 일치하는 모든 문자열 전역 검색
    • m
      • 행이 바뀌어도 패턴검색 계속

문자열 검색

  • RegExp 메서드와 함께 이용

임의의 문자열 검색

. 은 임의의 문자 한 개를 의미

반복 검색

{m,n}

  • 앞선 패턴이 최소 m 번, 최대 n 번 반복되는 문자역
  • , 뒤 공백있으면 정상동작 하지 않는다

{n}

  • {n,n}과 동치

{n,}

  • 앞선 패턴이 최소 n번 이상 반복

+

  • {1,}와 동치

?

  • 앞선 패턴이 최대 한 번 이상 반복되는 문자열
  • {0,1} 과 동치

OR 검색

|

  • or 의 의미를 가짐

[ ]

  • [] 내의 문자는 or로 동작한다.

-

  • 범위지정

알파벳 대소문자 미구별

숫자 검색

\d

  • 숫자를 의미
  • \D 는 숫자가 아닌 문자를 의미

\w

  • 알파벳, 숫자, 언더스코어
  • [A-Za-z0-9]와 동치
  • \W 는 \w와 반대로 동작

NOT 검색

  • […] 내의 ^은 not

시작 위치로 검색

  • […] 밖의 ^은 문자열의 시작을 의미한다.
  • […] 내의 ^ 은 not

마지막 위치로 검색

$

  • 문자열의 마지막
