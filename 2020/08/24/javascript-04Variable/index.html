변수

keyword : 메모리, 메모리 주소, 선언과 할당

변수

  • 왜 쓰이는가
    • 자바 스크립트 엔진이 코드를 평가할 때 코드의 의미를 알아야 한다.
  • 정의
    • 하나의 값을 저장하기 위해 확보한 메모리공간 또는 그 공간을 식별하기 위해 붙인 이름
    • 메모리 주소(값의 위치)를 가리키는 상징적 이름
  • 동작 원리
    1. 코드에서 값이 읽힐 때 값은 메모리 상의 임의의 위치에 저장
    2. 변수가 선언되면 메모리 공간을 확보하고 변수이름과 연결
    3. 값이 할당 될때 새로운 메모리 공간을 확보하고 할당 값을 새롭게 저장
    4. 가비지 컬렉터가 쓰레기 값들 청소
